# Forests Fire Prediction project 


## Install dependencies
```
cd <project_name>
npm install
```
## Run and debug
```
 nest start --debug  --watch
```

## MySQL prisma
### Pull from database
```
    npx prisma db pull
```
### Push to database 
```
    npx prisma db push
```
### Show data
```
    npx prisma studio
```
